# 开源与真实数据合规声明

作品：`industrial-cross-department-collaboration`

## 原创性

本 Skill 的核心指令、场景路由、输出模板、动作卡契约、测试用例、评分矩阵和评审材料均为本项目原创设计，围绕“工业现场信息到办公协同闭环”场景构建。

## 开源使用

当前 Skill 包内脚本仅使用 Python 标准库，不依赖第三方代码包。若后续引用开源项目、公开模板或第三方素材，必须补充：

- 来源链接。
- 开源协议。
- 使用范围。
- 与本 Skill 自研部分的差异说明。

核心 Skill 自研内容占比应始终保持不低于 50%。

## 真实数据

Skill 包和公开评审材料不得包含未脱敏真实客户数据、员工个人信息、系统账号、API 凭证、详细地址、合同价格、未公开订单信息或现场敏感记录。

如需使用真实或准真实案例，必须先执行脱敏：

```bash
python scripts/redact_input.py --text "<待脱敏记录>"
```

## 安全与授权边界

本 Skill 不替代授权人做以下判断：

- 质量放行。
- 安全许可。
- 危险作业批准。
- 设备维修最终根因判定。
- 外部系统实际同步状态。

对涉及安全、质量、客户交付和合规事项的输出，只生成事实包、待确认项、审批说明和协同动作卡。

## 部门协同表

| 部门 | 接手事项 | 输入依据 | 决策/授权边界 | 验收标准 | 截止时间 | 阻塞/升级 |
| --- | --- | --- | --- | --- | --- | --- |
| 生产部 | 牵头协调、确认现场执行和交期影响 | E | 生产现场协调 | 状态清晰、风险已同步 |  |  |
| 质量 | 隔离、复检、首件、放行建议 | E | 质量负责人授权 | 复检/首件结论明确 |  |  |
| 工程/工艺 | 参数、夹具、工艺版本、试产验证 | E | 工程变更流程 | 版本和验证记录完整 |  |  |
| 设备维修 | 故障检查、维修、试运行 | E | 维修负责人确认 | 试运行和维修记录完整 |  |  |
| EHS/安全 | 作业许可、风险隔离、安全宣导 | E | 安全员/资质人员确认 | 风险隔离和宣导闭环 |  |  |
| PMC | 排产、产能、交期、加班/插单评估 | E | 计划调整权限 | 交付影响和计划方案明确 |  |  |
| 仓库/采购 | 物料、包材、备件、外协资源 | E | 采购/仓储权限 | 到料时间和替代方案明确 |  |  |

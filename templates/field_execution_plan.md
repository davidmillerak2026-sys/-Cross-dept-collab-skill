# 现场执行编排模板

当输入涉及派工、维修执行、预测性维护、现场服务、备件申请或恢复生产时使用。

## 现场执行包

| 模块 | 内容 | 状态 |
| --- | --- | --- |
| 资产/设备 |  | 已确认/待确认 |
| 故障现象 |  | 已确认/待确认 |
| 业务影响 | 停机、良率、交付、质量隔离或安全影响 | 已确认/待确认 |
| 班组技能 | 需要的维修、电气、机械、工艺、质量、EHS 或外协技能 | 已确认/待确认 |
| 备件/工具 | 备件、工具、仪表、夹具、附件 | 已确认/待确认 |
| 作业许可 | 带电、动火、高处、化学品、锁定挂牌或其他许可 | 已确认/待确认 |
| 停机窗口 | 可维修窗口、生产协调、恢复时点 | 已确认/待确认 |
| 现场验证 | 检查点、测量项、试运行条件 | 已确认/待确认 |
| 验收标准 | 复检、试运行、质量确认、系统回执 | 已确认/待确认 |
| 失败回退 | 备件不到、权限不足、试运行失败、风险未隔离时的下一步 | 已确认/待确认 |

## 输出边界

- 没有验收证据时，不写“已修复”。
- 没有系统回执时，不写“已关闭系统记录”。
- 安全许可和质量放行必须等待授权角色确认。

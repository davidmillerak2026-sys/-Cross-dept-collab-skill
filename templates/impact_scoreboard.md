# 影响评分板模板

用于生产部牵头的跨部门异常，在维修、质量、安全、交付、物料和成本之间快速统一影响判断。它服务于管理决策，不替代 MES、QMS、EHS、SAP/ERP 或 PMC 的正式系统结论。

## 评分原则

- 只允许基于已确认事实打分；推断必须单独标记。
- 影响等级建议使用 `L0`、`L1`、`L2`、`L3`：
  - `L0`：当前无明显影响或尚未触发
  - `L1`：局部影响，可在班组或单部门内消化
  - `L2`：跨部门影响，已影响排产、质量隔离、安全许可或交付计划
  - `L3`：高风险或管理层必须介入，涉及停线扩大、客户承诺、安全升级或重大成本暴露
- 没有系统回执或实测数据时，不把估算写成已确认损失。

## 结构化字段建议

| field | 含义 | 示例 |
| --- | --- | --- |
| `downtime_minutes` | 已确认停线分钟数 | `18` |
| `production_loss_units` | 已确认少产数量或受影响箱数 | `32 箱待复检` |
| `isolated_quantity` | 质量隔离数量 | `240` |
| `affected_orders` | 受影响订单或客户数 | `SO-20260617-018` |
| `delivery_risk_level` | 交付风险等级 | `L2` |
| `material_eta` | 关键物料/备件到料时间 | `2026-06-17 18:00` |
| `cost_exposure_estimate` | 当前可审计的成本暴露估算 | `待确认` |
| `blocked_close_reason_count` | 当前阻止闭环的关键缺口数量 | `3` |
| `decision_due_by` | 管理决策最晚时间 | `2026-06-17 15:30` |

# 办公协同消息模板

## 工单同步

```text
【工单同步】{产线/设备} 于 {时间} 发生 {问题现象}，当前影响 {影响范围}。已采取措施：{已采取措施}。请 {责任方} 在 {截止时间} 前完成 {待办事项}，验收标准：{验收标准}。待确认：{待确认信息}。
```

## 质量异常同步

```text
【质量异常】{批次/SKU} 发现 {异常现象}，影响范围 {数量/区域}。建议先隔离并完成复检，未经质量授权不建议放行。请质量确认复检标准，生产/仓库配合追溯，销售暂缓对外承诺。
```

## EHS 整改同步

```text
【EHS整改】现场发现 {隐患描述}，风险等级 {优先级}。请先隔离风险并按 SOP 处理，由有资质人员确认整改结果。待办：{整改动作}。验收标准：{验收标准}。
```

## 会议复盘同步

```text
【复盘结论】本次复盘围绕 {主题}。已确认事实：{事实摘要}。决策点：{决策点}。行动项：{责任方+动作+截止时间}。待确认：{待确认信息}。
```
